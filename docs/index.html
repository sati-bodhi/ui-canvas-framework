Complete Development Workflow

Design Generation

Use SuperDesign VS Code extension to generate mockups from natural language prompts

"Create a mobile dashboard for garden task management with card-based layout"

Canvas Staging

Framework automatically stages SuperDesign mockups for development iteration

npx ui-canvas canvas stage garden-dashboard

Component Development

Break mockups into reusable web components following three-layer architecture

npx ui-canvas generate component task-card --variant=compact

Visual Testing

Automated Playwright tests ensure components match approved designs

npm run test:visual --update-snapshots

Production Deployment

Components are production-ready with enforced consistency and automated testing

npm run build && npm run deploy

Quick Start Documentation

Installation

npm install @ui-canvas/framework
npx ui-canvas init my-project

Configuration

// ui-canvas.config.js export default { framework: 'vanilla', layers: { components: 'components', pages: 'pages', workflows: 'workflows' } }

CLI Commands

ui-canvas serve Start development server
ui-canvas canvas stage <mockup> Stage SuperDesign mockup
ui-canvas validate Check architecture compliance
